1. Создайте Telegram-бота и получите API Key. Откройте @BotFather, создайте нового бота, придумайте имя и username (заканчивается на bot). В ответ BotFather пришлёт токен — это и есть Bot API Key, сохраните его.
  2. Узнайте Chat ID нужного канала или чата. Добавьте бота @re_GetMyIDBot в нужный чат/канал или просто перешлите ему сообщение из нужного канала/чата. В ответ бот пришлёт Chat ID идентификатор — используйте его в приложении.
  3. Вставьте токен и chat_id в интерфейс приложения. При желании включите опцию «Сохранять токен и chat_id», чтобы сохранить введенные данные.
    Важно: данные будут сохранены только на этом устройстве.
  4. Дождитесь проверки доступа. Произойдет автоматическая проверка бота и чата/канала, на возможность отправки сообщения. После успешной проверки появятся карточки с именем и аватаром, а кнопка «Создать пост» активируется.
  5. Подготовьте содержимое поста. Загрузите локальный Markdown/HTML-файл или вставьте ссылку на файл README.md в GitHub или Markdown/HTML любом другом ресурсе (при условии открытого доступа к содержимому).
  6. Нажмите «Создать пост». Произойдет отправка поста в указанный чат или канал.
  7. Проверьте результат работы в Telegram. Все сетевые вызовы к Telegram выполняются на сервере, а учётные данные сохраняются только локально на устройстве пользователя, если вы включили соответствующий флажок.

Напоминание: внимательно относитесь к безопасности токена бота. При необходимости отзовите его через BotFather командой /revoke.
